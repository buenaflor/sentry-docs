description: "Learn about Sentry's Redux integration."

_(Available in version 1.8.0 and above)_

Redux support is included in the `@sentry/react-native` package since version `1.8.0`. To apply Sentry to Redux, use `Sentry.createReduxEnhancer` at the same place that you initialize your Redux store.
